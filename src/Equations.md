## Electron Inertial Length
$\delta_e = \frac{c}{\omega_{p_e}}$

Plot $\delta_{e,i}$ as a function of density $n_{e,i}$

```
| Object   | n_p [cm^-3] | 
|----------|-------------|
| Mercury  | 40          | 
| Moon     |  5          | 
| Mars     |  2          | 
| Ganymede |  8          | 
| Europa   | 30          | 
```

## Charge Density
$\rho=qn \Rightarrow n = \frac{q}{\rho}$
